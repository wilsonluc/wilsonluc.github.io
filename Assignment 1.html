Project Idea: Sudoku Solver

Something related to mobile app dev

Overview: Sudoku is a puzzle where players insert numbers into nine 3x3 grids connected to each other, forming a 9x9 grid in total. The goal is to fill the Sudoku puzzle with number in a way such that every number appears exactly once in each horizontal line, vertical line, and square. The goal is to create a program to automatically solve a Sudoku puzzle given a valid input. It will only be able to handle a standard 9x9 Sudoku format. The user can be asked if they want the problem to be solved through bruteforce or pre-existing Sudoku techniques. Each fixed number in the Sudoku puzzle must be input by the user.

Motivation: Ever since I was first introduced to Sudoku as a child by my father, I've been fascinated by the concept that most unique Sudoku puzzles have only one specific solution. What has impressed me even more than this is the fact that more than six sextillion (6.67*1021) unique combinations exist in a standard Sudoku puzzle. Although so many unique Sudoku puzzles exist, there are websites that have the ability to take these and solve them in almost no time at all. As a result, this has inspired me to challenge my programming knowledge to create something that can rival these websites in terms of speed.
